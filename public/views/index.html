Work

Füdr

Füdr is a site I created myself. It is an app to help decide where to eat in south Minneapolis. It pulls a random dish from a MongoDB database that I have assembled and allows the user to choose if the dish is something they might like to eat. Restaurant information is pulled from Factual API. I would suggest not looking at this if you're hungry. Actually, look at it even if you are :)

American Drapery Systems

American Drapery Systmes asked us to build a web application to help input, keep track of, and manipulate measurement data when doing a survey for a client. I did a lot of the front-end work, setting up input fields and moving the data to the appropriate routes, along with styling.

Fun Things

Fun Things web application was created in one afternoon with a group. I did most of the front end work on this app. The trickiest part was getting the swipe to work on the cards. I used an already created directive called ng-swippy. Anyone can add "fun things" so there may be some odd ones.
