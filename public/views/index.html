Füdr

AngularJS, Angular Material, HTML5, CSS, Node.js, Express.js, MongoDB, Firebase

Füdr is a site I created myself. It is a web application to help decide where to eat in south Minneapolis. A random dish is displayed from a database I assembled and allows the user to choose if the dish is something they might like to eat. Restaurant information is pulled from Factual API. I would suggest not looking at this if you're hungry but actually, look at it even if you are :)
