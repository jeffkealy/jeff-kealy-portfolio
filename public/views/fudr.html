Home

FüDr is an app I created, then recreated twice more. This is version 3, with 4 on the way. I wanted to create tinder like swipe style app to help decide where to eat. The user swipes through dishes and the look though their right swipes.

Add Dish

On the admin side, a dish can be added which first uses the yelp api to look up the restaurant, upload a photo, and add the proper criteria to add a dish.
